02 Tor Browser

Скачивайте только с torproject.org. Уровень безопасности — «Безопасный». Используйте мосты (obfs4, Snowflake) при блокировках.

❌ Не разворачивайте на весь экран, не входите в личные аккаунты, не ставьте расширения.
03 Анонимные браузеры
LibreWolf (⭐⭐⭐⭐⭐) — максимальная приватность из коробки
Brave (⭐⭐⭐⭐) — Shields + режим Tor
Firefox Hardened — about:config + arkenfox
Chrome / Edge — ❌ не использовать
04 Tails OS

Загружается с флешки, не оставляет следов, весь трафик через Tor. Идеально: Tails + публичный WiFi.

💡 Для высокорисковых операций — только Tails.
05 VeraCrypt + метаданные
VeraCrypt — шифрование диска, скрытые тома
MAT2 / ExifTool — удаление EXIF (GPS, модель камеры)
06 ОПСЕК — поведение
  • ❌ Не использовать старые ники и клички
  • ❌ Не смешивать анонимную и реальную личность
  • ❌ Не рассказывать о своих инструментах анонимности
  • ✅ Создайте легенду, используйте менеджер паролей
07 Аппаратная анонимность
  • Смена MAC-адреса (macchanger, Tails делает это автоматически)
  • Покупка ноутбука и роутера за наличные
  • Отключение Bluetooth, TPM, Intel AMT в BIOS
08 Windows
  • O&O ShutUp10++ — отключает телеметрию
  • WPD — блокирует IP-адреса Microsoft
  • SimpleWall — файрвол
  • Локальная учётная запись (без Microsoft Account)
После крупных обновлений Windows настройки слетают — применяйте заново.
09 Android
  • Лучший вариант: GrapheneOS (требует Pixel) или CalyxOS
  • Используйте microG вместо Google Play Services
  • Магазин приложений — F-Droid
  • Отключить геолокацию, Wi-Fi/Bluetooth-сканирование, диагностику
10 iOS / iPhone
  • Onion Browser + Orbot (Tor на iPhone)
  • Отключить аналитику и персонализированную рекламу
  • Не использовать iCloud для важных данных
  • Полной анонимности на iOS нет из-за ограничений Apple
11 macOS
  • Little Snitch — файрвол
  • Отключить аналитику, Siri, iCloud Drive, Find My Mac
  • Терминал: sudo launchctl unload -w /System/Library/LaunchDaemons/com.apple.analyticsd.plist
12 Публичные WiFi
  • Включайте VPN ДО подключения к WiFi
  • Меняйте MAC-адрес перед подключением
  • Не используйте домашний WiFi для анонимных сессий
13 Onion-ссылки

Поиск: Ahmia, Torch, DuckDuckGo .onion. Официальные onion: Facebook, ProtonMail, NYT, BBC.

⚠️ Сравнивайте длинные адреса по буквам — фишеры меняют одну букву.

  • ❌ Никогда не диктовать код из СМС
  • ❌ Не переходить по ссылкам из писем
  • ❌ Не открывать .exe, .scr, .js, .vbs
  • ✅ Звонок из банка — положить трубку и перезвонить самому

15 RAT — удалённый доступ

RAT — вирус, дающий хакеру полный контроль над ПК.

Что делать:

  1. 1. Отключить интернет
  2. 2. Сохранить важные файлы на чистую флешку
  3. 3. Полная переустановка Windows
  4. 4. Сменить ВСЕ пароли
  5. 5. Отозвать все активные сессии
⚠️ Антивирус не гарантирует удаление RAT. Только чистая переустановка.
16 Вирусы
⛏️ МАЙНЕРЫ

CPU на 100% в простое. Лечение: uBlock Origin, Malwarebytes.

🎭 ТРОЯНЫ

Неизвестные программы. Лечение: переустановка Windows.

🔒 ВИНЛОКЕРЫ

🕵️ СТИЛЕРЫ

18 Безопасные пароли

  • Пример: кошка-лампа-гвоздь-93!@
  • Менеджеры: Bitwarden (облачный), KeePassXC (локальный)
  • Не делать: хранить пароли в браузере, использовать один пароль везде
⚠️ Включите 2FA через аутентификатор (Aegis, Google Authenticator).

22 🔥 Как стать полностью анонимным
МАКСИМАЛЬНАЯ ЗАЩИТА

Полная анонимность — это система, а не один инструмент.

1. ЦИФРОВАЯ ЛИЧНОСТЬ
  • Создайте легенду — имя, возраст, профессия, хобби
  • Не используйте старые никнеймы
  • Разные аккаунты для разных задач
2. СЕТЕВАЯ ИЗОЛЯЦИЯ
  • VPN + Tor (Mullvad → Tor)
  • Публичный WiFi (кофейни, библиотеки)
  • Смена MAC-адреса
  • Tails для важных задач
3. ПОВЕДЕНИЕ
  • Не рассказывайте о своей анонимности
  • Меняйте время выхода в сеть
  • Ноутбук за наличные
⚠️ Даже один разрыв в легенде может вас деанонимизировать.
4. ТЕХНИЧЕСКИЕ СРЕДСТВА
  • Mullvad (оплата наличными)
  • Tor Browser (без расширений)
  • Tails для важных задач
  • VeraCrypt, менеджер паролей, 2FA, бекапы
📌 Это минимальный набор для 95% анонимности.
5. ФИЗИЧЕСКАЯ БЕЗОПАСНОСТЬ
  • Избегайте камер при покупке техники
  • Платите наличными
  • Уничтожайте SIM-карты
  • Маскировка
⚠️ Анонимность в сети не имеет смысла, если вас могут найти в реальной жизни.

24 🔐 PGP / GPG — шифрование
КРИПТОГРАФИЯ

PGP (Pretty Good Privacy) — стандарт шифрования для сообщений и файлов.

  • Установка: Gpg4win (Win), GPG Suite (Mac), sudo apt install gnupg (Linux)
  • Генерация ключа: gpg --full-generate-key (RSA 4096, срок 2 года)
  • Экспорт публичного ключа: gpg --export --armor [email] > public-key.asc
  • Шифрование: gpg --encrypt --armor --recipient [email] file.txt
  • Расшифровка: gpg --decrypt file.asc
⚠️ Никогда не передавайте секретный ключ. Храните его в надёжном месте.
25 🧅 Whonix
ВИРТУАЛЬНАЯ ЗАЩИТА

Whonix — две виртуальные машины: Gateway (шлюз) и Workstation. Весь трафик через Tor.

  • Gateway — подключается к Tor
  • Workstation — рабочая среда
  • Изоляция: даже при взломе Workstation IP не узнают
📌 Whonix тяжелее Tails, но даёт лучшую изоляцию.
26 🔒 Qubes OS
МАКСИМАЛЬНАЯ ИЗОЛЯЦИЯ

Qubes OS — операционная система, построенная на виртуализации. Каждое приложение работает в отдельной виртуальной машине.

  • Изоляция: каждый домен (приложение) изолирован
  • Безопасность: взлом одного домена не влияет на другие
  • Для кого: для параноиков, журналистов, диссидентов
💡 Qubes OS — один из самых безопасных способов работы в сети.
27 👤 Физическая безопасность
РЕАЛЬНАЯ ЖИЗНЬ

Анонимность в сети не имеет смысла, если вы раскрываете себя в реальной жизни.

  • Камеры: избегайте камер при покупке техники, SIM-карт
  • Наличные: всегда платите наличными
  • Уничтожение: уничтожайте SIM-карты, флешки, бумажные записи
  • Маскировка: меняйте внешность (одежда, очки, походка)
  • Перемещения: не используйте свой транспорт, меняйте маршруты
🔴 Если вас могут найти в реальной жизни — анонимность в сети бесполезна.
